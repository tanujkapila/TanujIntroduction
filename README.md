# Brochure-style Resume — Tanuj Kapila

This is a premium, brochure-style single-page site built with GSAP animations and a pinned horizontal storytelling section.

## How to deploy

1. Create a repository named `resume` in your GitHub account (or use any repo).
2. Upload the contents of this folder to the repo root.
3. Commit and push to the `main` branch.
4. Enable GitHub Pages (Settings → Pages) and set source to `main` branch / root.
5. Your site will be available at `https://<your-username>.github.io/resume/`.

## Customize
- Replace the hero background in `assets/styles.css` (hero-bg background-image) with your photo.
- Edit copy in `index.html` to personalise your journey, experience and fun facts.
- Replace the mailto in the CTA with your preferred contact method.
